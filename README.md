# Vite React Template

## About

A clean Vite React template with:

- [**Vite ^4**](https://vitejs.dev/)
- [**React ^18**](https://react.dev/)
- [**Typescript ^5**](https://www.typescriptlang.org/)
- [**Tailwind ^3**](https://tailwindcss.com/)
- [**ESlint ^8**](https://eslint.org/)
- [**Prettier ^3**](https://prettier.io/)

## Usage

Copy the project template using [degit](https://github.com/Rich-Harris/degit):

```
npx degit rbika/vite-react-template <project-name>
```

Change directory and install the dependencies:

```
cd <project-name> && npm install
```

Run the local server:

```
npm run dev
```
